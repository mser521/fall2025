## ğŸ› Troubleshooting

### Common Issues

1. **Styles not applying**: Check file paths in `<link>` tags
2. **JavaScript not working**: Verify script is linked before `</body>`
3. **Images not loading**: Check image URLs are correct
4. **Layout broken**: Verify CSS Grid/Flexbox syntax
5. **Animations not triggering**: Check data attributes are correct
